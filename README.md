An example of writing erlang-irc-bot plugins in [Elixir](http://elixir-lang.org/)
=================================================================================

To compile:

    mix deps.get
    mix compile


Then copy `settings.cfg.example` to `settings.cfg`, edit it to set passwords and nickname, and run it:

    ./run.sh
